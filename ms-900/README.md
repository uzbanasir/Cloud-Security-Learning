# MS-900: Microsoft 365 Fundamentals

## Course Summary

MS-900: Microsoft 365 Fundamentals is a beginner-level course that introduces Microsoft 365 cloud services, productivity tools, collaboration solutions, security concepts, compliance features, licensing, and support options.

This course is useful for understanding how organizations use Microsoft 365 for email, file sharing, communication, teamwork, device management, identity, security, and compliance.

> Note: Microsoft retired the MS-900 exam on March 31, 2026, but the course knowledge is still useful for understanding Microsoft 365 environments.
