# AZ-900: Microsoft Azure Fundamentals

## Course Summary

AZ-900: Microsoft Azure Fundamentals is a beginner-level course that introduces the core concepts of cloud computing and Microsoft Azure. It is designed for learners who are new to Azure and want to understand cloud services, Azure architecture, security, governance, pricing, and basic management tools.

This course builds foundational knowledge for future Azure and cybersecurity certifications, including AZ-104, AZ-500, SC-900, and SC-200.

## Main Topics Covered

### 1. Cloud Concepts

This section explains what cloud computing is and why organizations use cloud services.

Key topics include:

- Cloud computing basics
- Public, private, and hybrid cloud models
- Shared responsibility model
- Consumption-based pricing
- High availability
- Scalability
- Reliability
- Elasticity
- Manageability
- IaaS, PaaS, and SaaS

### 2. Azure Architecture and Services

This section introduces the main building blocks of Microsoft Azure.

Key topics include:

- Azure regions
- Region pairs
- Availability zones
- Azure datacenters
- Resource groups
- Azure subscriptions
- Management groups
- Azure compute services
- Virtual machines
- Containers
- Azure Functions
- Azure App Service
- Azure Virtual Network
- Azure storage services
- Azure databases

### 3. Azure Management and Governance

This section explains how Azure resources are managed, secured, monitored, and controlled.

Key topics include:

- Microsoft Entra ID
- Role-Based Access Control
- Azure Policy
- Resource locks
- Tags
- Cost Management
- Azure Advisor
- Azure Monitor
- Service Health
- Azure portal
- Azure PowerShell
- Azure CLI
- Azure Resource Manager templates
