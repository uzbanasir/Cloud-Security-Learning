# AZ-900: Microsoft Azure Fundamentals

## First Hands-On Lab: Creating a Virtual Machine

The first hands-on lab focused on creating a Virtual Machine in Microsoft Azure. This lab introduced the basics of deploying a cloud-based compute resource and helped me understand how Azure resources are created and managed through the Azure Portal.

## Lab Activities

- Opened the Azure Portal
- Started the process of creating a Virtual Machine
- Selected basic VM settings such as subscription, resource group, region, image, and size
- Reviewed administrator account options
- Reviewed networking settings for the virtual machine
- Reviewed and created the VM resource
- Learned how Azure organizes resources during deployment

## Skills Practiced

- Azure Portal navigation
- Creating a cloud compute resource
- Understanding resource groups
- Understanding Azure regions
- Reviewing VM configuration settings
- Basic Azure resource deployment
- Cloud infrastructure fundamentals
