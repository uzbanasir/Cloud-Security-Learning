# AZ-104: Microsoft Azure Administrator

## Course Summary

AZ-104: Microsoft Azure Administrator is an intermediate-level Azure course focused on managing, configuring, securing, and monitoring Microsoft Azure environments.

This course builds on AZ-900 by moving from basic cloud concepts into real Azure administration tasks. It is designed for learners who want to manage Azure resources, identities, storage, compute, networking, governance, and monitoring in a professional cloud environment.

AZ-104 is an important course for anyone pursuing cloud security, cybersecurity, SOC Analyst, or Azure administrator roles because it teaches how Azure environments are actually managed in real organizations.

## Main Topics Covered

### 1. Manage Azure Identities and Governance

This section focuses on managing access, users, permissions, subscriptions, and governance in Azure.

Key topics include:

- Microsoft Entra ID users and groups
- Role-Based Access Control
- Azure subscriptions
- Resource groups
- Management groups
- Azure Policy
- Resource locks
- Tags
- Cost management
- Azure Advisor

### 2. Implement and Manage Storage

This section explains how to create, configure, secure, and manage Azure storage services.

Key topics include:

- Storage accounts
- Azure Blob Storage
- Azure Files
- Storage redundancy
- Access keys
- Shared Access Signatures
- Storage firewalls
- Storage tiers
- Soft delete
- Blob lifecycle management
- Azure Storage Explorer
- AzCopy

### 3. Deploy and Manage Azure Compute Resources

This section focuses on creating and managing cloud compute resources.

Key topics include:

- Azure Virtual Machines
- VM sizes and images
- Availability options
- Virtual machine extensions
- Azure App Service
- Containers
- Azure Kubernetes Service basics
- Azure Container Instances
- Backup and recovery for compute resources

### 4. Implement and Manage Virtual Networking

This section introduces Azure networking concepts and how resources communicate securely.

Key topics include:

- Virtual networks
- Subnets
- Private and public IP addresses
- Network Security Groups
- Route tables
- Azure DNS
- VPN Gateway
- VNet peering
- Load balancers
- Application Gateway
- Network troubleshooting

### 5. Monitor and Maintain Azure Resources

This section explains how administrators monitor Azure environments and keep resources healthy.

Key topics include:

- Azure Monitor
- Log Analytics
- Alerts
- Metrics
- Activity logs
- Resource health
- Service health
- Backup
- Recovery Services vaults
- Azure Advisor recommendations
